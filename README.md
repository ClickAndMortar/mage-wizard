# Mage Wizard

## Requirements

- [Node.js](https://nodejs.org/en/) (`18.x` or newer)
- [pnpm](https://pnpm.io/) (v6 or newer)

## Setup

```bash
pnpm install
```

## Development server

Start the development server on `http://localhost:3000`:

```bash
pnpm run dev
```

## Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```
